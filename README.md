# AppleSurvival
OVERVIEW: apple survival game that im making

WHAT IT DOES: This is a simple text-based survival game where you go place to place in search of weapons to slay a dragon. You must be careful, however as if the dragon encounters you it will attack. You lose if you run out of health, you win if you kill the dragon. Score is based on turns survived, and you get +100 if you kill the dragon and win.

WHY: This is a game made from my girlfriend's suggestion, and it will be a Valentines gift for her.

DIFFICULTIES WHILE CODING: There were many difficulties, like trying to use the apple and dragon objects throughout the numerous interconnected aid functions, and thinking of things to add was another struggle.

HOW TO RUN IT: Download the files, run main.py in your terminal or an IDE to start.
